# MindAura (TFG)

Backend desarrollado como Trabajo de Fin de Grado.

## Tecnologías

- Node.js / Express
- MongoDB

## Configuración

1. Copia `.env_example` a `.env`:
   ```bash
   cp .env_example .env

2. Completa tus credenciales y SECRET_KEY.
3. Instala dependencias:
   ```bash
   npm install
4. Arrancar el servidor:
   ```bash
   npm start
